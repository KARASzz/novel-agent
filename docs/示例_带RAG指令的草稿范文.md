# 第一集：龙王隐忍，赘婿受辱

[场景：林家豪华别墅，夜晚，内]
[人物：林震（男主/隐藏龙王）、苏婉儿（女主/林家千金）、苏老太（林家家主）、赵公子（反派配角）]

[剧情]
林震端着洗脚水跪在苏老太面前，被一脚踹翻。洗脚水泼了一地。
苏老太冷笑：“废物！今天婉儿要是跟赵公子签了合同，你也该滚出我们林家了！”
赵公子在一旁得意洋洋地搂住苏婉儿的肩膀：“老太太说得对，这废物配不上婉儿。婉儿你看，这是我送你的三千万翡翠项链。”

苏婉儿看着林震，眼神复杂，既有失望又有一丝不忍。
林震低头收拾地上的水渍，眼神隐忍，没有反驳。

就在这时，林震的破旧手机突然震动。他偷偷拿出来一看。
短信显示：【龙王殿十万将士已集结完毕，只等您一声令下，踏平燕京！】

[钩子]
林震放下手机，抬头看向赵公子，嘴角勾起一抹冷冽的笑意：
“三千万的破石头也敢拿出来丢人现眼？明天，我会让整个燕京知道，谁才是真正配得上婉儿的人！”

旁白：“隐忍三年，龙王觉醒！明日的宴会，必将天翻地覆！”
