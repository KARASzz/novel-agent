# 案例库 (CaseLibrary) 知识仓

> 本目录存放所有与**已上线短剧的成功/失败拆解分析**相关的案例文档。

## 目录下应该放什么？

1. **爆款剧拆解**：对标竞品的结构、卡点布局、钩子设计的逐集深度复盘。
2. **失败案例分析**：审核打回、数据扑街的项目反思总结与避坑经验。
3. **同题材竞品对比**：相似赛道内不同作品的差异化策略与数据表现对比。

## 格式示例

```markdown
# 爆款拆解：《XX归来》

- **题材**：男频战神
- **总集数**：80集
- **首日数据**：播放量 XXX，付费转化率 XX%
- **前3集钩子设计**：……
- **第15集付费卡点**：……
- **成功核心要素**：……
```
